# Guia de Estilo Front-end

## Layout

Os designs foram criados para as seguintes larguras:

- Mobile: 375px
- Desktop: 1440px

> 💡 Estas são apenas as dimensões do design. Garanta que o conteúdo seja responsivo e atenda aos requisitos  Web Content Accessibility Guidelines (WCAG) testando em toda a gama de tamanhos de tela, de 320px até telas grandes.
